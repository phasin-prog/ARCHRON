# ARCHRON Sitemap & Dynamic Route Architecture v3.0

**Version:** 3.0  
**Status:** Living Architectural Map  
**Framework:** Next.js 16 App Router (RSC, Server Actions, ISR, SSG, Dynamic Routes)  
**Language Strategy:** Thai-First Architecture (อธิบายโครงสร้างเป็นภาษาไทยควบคู่กับเส้นทาง URL มาตรฐานสากล)

---

## 🧭 คำชี้แจงสถาปัตยกรรม (Architectural Overview)

เอกสารนี้รวบรวมแผนผังเส้นทาง (Route Map) ทั้งหมดในระบบ ARCHRON v3.0 ครอบคลุมตั้งแต่หน้าสาธารณะ โครงข่ายความรู้ (Knowledge Graph/Constellation) ศูนย์รวมการสืบค้น พื้นที่ทำงานนักเขียน (Studio) จนถึงระบบเครือข่ายและ API เส้นทางทั้งหมดถูกออกแบบตามหลัก **Knowledge Ontology** เพื่อรองรับการสืบค้นความสัมพันธ์เชิงลึกอย่างไร้รอยต่อ

---

## 1. ROOT (เส้นทางระดับรากฐาน)

ศูนย์รวมจุดเริ่มต้นและเครื่องมือค้นพบความรู้เบื้องต้น:

```text
/
├── explore         (สำรวจความรู้ตามกระแสและหัวข้อแนะนำ)
├── discover        (ค้นพบความรู้ใหม่แบบสุ่มและตามหมวดหมู่)
├── search          (ศูนย์ค้นหากลางแบบครอบคลุม)
├── constellation   (แผนที่ความรู้แบบรัศมีโต้ตอบได้)
├── timeline        (เส้นเวลาประวัติศาสตร์ทางปัญญา)
└── compare         (เปรียบเทียบมโนทัศน์/นักคิด/สำนักคิด)
```

## 2. KNOWLEDGE (คลังความรู้ปฐมภูมิและทุติยภูมิ)

เส้นทางเนื้อหาหลัก จัดเรียงตามลำดับชั้นความรู้ (Knowledge Chain Ontology):

```text
/articles                       (รายการบทความวิเคราะห์ทั้งหมด)
/articles/[slug]                (หน้าอ่านบทความเดี่ยว — SSG + ISR 5m)

/concepts                       (รายการมโนทัศน์และแนวคิดสำคัญ)
/concepts/[slug]                (หน้าอ่านแนวคิดปฐมภูมิและ Backlinks)

/thinkers                       (ทำเนียบนักคิดและนักปราชญ์)
/thinkers/[slug]                (หน้าประวัตินักคิดเชิงวิชาการชั้นสูง)

/schools                        (สำนักคิดและประเพณีทางปัญญา)
/schools/[slug]                 (หน้าเจาะลึกสำนักคิดและข้อถกเถียงหลัก)

/theories                       (ทฤษฎีและกรอบแนวคิดการอธิบาย)
/theories/[slug]                (หน้าอ่านรายละเอียดทฤษฎี)

/books                          (คลังบรรณานุกรมและผลงานชิ้นเอก)
/books/[slug]                   (หน้าข้อมูลหนังสือและบทวิจารณ์)

/references                     (เอกสารอ้างอิงและหลักฐานปฐมภูมิ)
/references/[slug]              (หน้าแสดงต้นฉบับอ้างอิง)

/timelines                      (รายการเส้นเวลาประวัติศาสตร์เฉพาะด้าน)
/timelines/[slug]               (หน้าแสดงเส้นเวลาเชิงโต้ตอบ)

/collections                    (ชุดความรู้และซีรีส์การอ่านที่คัดสรร)
/collections/[slug]             (หน้าอ่านชุดความรู้ต่อเนื่อง)

/topics                         (หัวเรื่องหลักทางมนุษยศาสตร์)
/topics/[slug]                  (รวมเนื้อหาตามหัวเรื่อง)

/tags                           (ป้ายกำกับและคำสำคัญข้ามศาสตร์)
/tags/[slug]                    (รวมเนื้อหาตามป้ายกำกับ)

/languages                      (รากภาษาและนิรุกติศาสตร์)
/languages/[slug]               (คำศัพท์และมโนทัศน์ตามรากภาษา)

/symbols                        (สัญลักษณ์และจิตวิทยาสัญลักษณ์)
/symbols/[slug]                 (การตีความสัญลักษณ์ตามหลักวิชาการ)

/mythologies                    (ปรัมปราวิทยาและตำนานมนุษย์)
/mythologies/[slug]             (หน้าเล่าเรื่องและวิเคราะห์ตำนาน)

/religions                      (ศาสนศาสตร์และประเพณีความเชื่อ)
/religions/[slug]               (การเปรียบเทียบและวิเคราะห์ปรัชญาศาสนา)

/civilizations                  (อารยธรรมและประวัติศาสตร์สังคม)
/civilizations/[slug]           (พัฒนาการทางความคิดแต่ละอารยธรรม)
```

## 3. KNOWLEDGE GRAPH (กราฟความรู้และเครือข่ายความสัมพันธ์)

```text
/graph                          (ภาพรวมกราฟความรู้ทั้งระบบ)
/graph/concepts                 (เครือข่ายความเชื่อมโยงมโนทัศน์)
/graph/thinkers                 (เครือข่ายอิทธิพลระหว่างนักคิด)
/graph/schools                  (สายธารและสาขาสำนักคิด)
/graph/theories                 (การต่อยอดและขัดแย้งของทฤษฎี)
/graph/books                    (ความเชื่อมโยงของวรรณกรรมและตำรา)
/graph/history                  (กราฟวิวัฒนาการตามช่วงเวลาประวัติศาสตร์)
/graph/network                  (เครือข่ายความสัมพันธ์เชิงความหมายระดับลึก)
```

```text
/opensearch.xml                 (ค้นหา ARCHRON โดยตรงผ่านแถบที่อยู่เบราว์เซอร์)
```

```text
/concepts/category/[slug]       (กรองแนวคิดตามสาขาวิชา)

/thinkers/school/[slug]         (กรองนักคิดตามสำนักคิด)
/thinkers/era/[slug]            (กรองนักคิดตามยุคสมัยประวัติศาสตร์)

/books/author/[slug]            (กรองตำราตามผู้แต่ง)
```
