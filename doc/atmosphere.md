## Sensory Brand (แบรนด์เชิงสัมผัส)

### Visual Touch (สัมผัสทางสายตา)

- **Soft Transitions:** การเปลี่ยนแปลงสีและแสงค่อยๆ ไล่ระดับ ไม่กระชาก
- **Gentle Shadows:** เงาอ่อนนุ่ม ไม่แข็งกระด้าง
- **Paper Texture:** ลวดลายกระดาษโบราณ (opacity: 2-4%)
- **Warm Glow:** แสงเรืองรองเฉพาะจุด (opacity: 4-6%)

### Temporal Touch (สัมผัสเชิงกาลเวลา)

- **Slow Ambient Motion:** การเคลื่อนไหวพื้นหลังช้าๆ (8s cycle)
- **Gentle Hover States:** สถานะเมื่อวางเมาส์ค่อยๆ ปรากฏ
- **Patient Loading:** การโหลดที่อดทน ไม่รีบร้อน
- **Timeless Typography:** ตัวอักษรคลาสสิกที่ไม่ล้าสมัย

### Spatial Touch (สัมผัสเชิงพื้นที่)

- **Breathing Room:** ระยะห่างโปร่งสบาย ไม่บีบอัด
- **Hierarchical Depth:** ลำดับชั้นลึกซึ้ง ไม่แบนราบ
- **Museum Spacing:** ระยะห่างแบบพิพิธภัณฑ์ ให้คุณค่าทุกองค์ประกอบ
- **Reading Column:** ความกว้างอ่านสบาย ~60-75 ตัวอักษร/บรรทัด

## Atmosphere Implementation Checklist

### Global Atmosphere

- [ ] พื้นหลัง: midnight blue (#080B16) ไม่ใช้ pure black
- [ ] ตัวอักษร: warm ivory (#F4F1EA) ไม่ใช้ pure white
- [ ] accent: gold (#C79A4A) สำหรับ brand elements
- [ ] shadows: soft, layered ไม่แข็ง
- [ ] transitions: slow, gentle ไม่กระชาก
- [ ] textures: paper, noise 2-4% opacity

### Context-Specific Atmosphere

- [ ] Psychology: reflective, organic, introspective
- [ ] Philosophy: contemplative, balanced, geometric
- [ ] History: nostalgic, layered, museum-like
- [ ] Religion: sacred, illuminated, symbolic
- [ ] Mythology: epic, textured, narrative
- [ ] Science: precise, clean, structured

### Sensory Details

- [ ] Ambient motion: 8s cycle, gentle
- [ ] Hover states: soft transitions
- [ ] Loading states: patient, not rushed
- [ ] Focus states: clear, visible, elegant
- [ ] Error states: informative, not alarming
