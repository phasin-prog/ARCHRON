## Tone of Voice (น้ำเสียง)

### หลักการ

> **"เป็นทางการ อบอุ่น สุขุม ชวนขบคิด ไม่ใช้ภาษาการตลาดฉาบฉวย"**

| ลักษณะ | คำอธิบาย | ตัวอย่าง |
|---|---|---|
| **Formal** | เป็นทางการ น่าเชื่อถือ ไม่หลุดลุ่ย | "แนวคิดพื้นฐานของจิตวิทยาวิเคราะห์" |
| **Warm** | อบอุ่น ไม่เย็นชา ไม่ห่างเหิน | "เชิญค้นหาความรู้ที่น่าสนใจ" |
| **Calm** | สุขุม ไม่เร่งรีบ ไม่ตื่นเต้นเกินไป | "ค่อยๆ ทำความเข้าใจทีละส่วน" |
| **Intellectual** | ชวนขบคิด ลึกซึ้ง ไม่ตื้นเขิน | "คำถามที่ท้าทายความเข้าใจเดิม" |
| **Non-marketing** | ไม่ใช้ภาษาการตลาดฉาบฉวย | หลีกเลี่ยง "สุดยอด!", "ดีที่สุด!", "ฟรี!" |

### หลีกเลี่ยง

| สิ่งที่ห้าม | ทำไม | ตัวอย่างที่ผิด |
|---|---|---|
| **Clickbait** | ไม่ซื่อสัตย์ต่อผู้อ่าน | "คุณจะไม่เชื่อสิ่งนี้!" |
| **Superlatives** | เกินจริง ไม่น่าเชื่อถือ | "ดีที่สุดในโลก!" |
| **Urgency** | สร้างความกดดันไม่จำเป็น | "สมัครเดี๋ยวนี้ก่อนหมดอายุ!" |
| **Informal** | ไม่เหมาะสมกับเนื้อหาวิชาการ | "ฮาาา รู้ยัง?" |
| **Technical jargon** | เข้าใจยากโดยไม่จำเป็น | ใช้คำศัพท์เทคนิคโดยไม่อธิบาย |

## Empty States (สถานะว่าง)

### ค้นหาไม่พบ

```text
ไม่พบผลลัพธ์สำหรับ "{query}"

ลองค้นหาด้วยคำอื่น หรือสำรวจแนวคิดที่เกี่ยวข้อง
```

### ยังไม่มีเนื้อหา

```text
ส่วนนี้กำลังอยู่ระหว่างการพัฒนา

ความรู้ใหม่กำลังจะมาถึงในเร็วๆ นี้
```

### หน้าว่าง

```text
ไม่มีข้อมูลในส่วนนี้

探索ความรู้อื่นๆ ใน ARCHRON
```

### ข้อผิดพลาด

```text
เกิดข้อผิดพลาดในการโหลดข้อมูล

กรุณาลองใหม่อีกครั้ง หรือติดต่อทีมงานหากปัญหายังคงอยู่
```

## Thai Writing Rules (กฎภาษาไทย)

### วรรคตอน

- ใช้ " " (เครื่องหมายอัญประกาศ) สำหรับคำพูด/ข้อความอ้างอิง
- ใช้ ' ' (เครื่องหมายอัญประกาศเดี่ยว) สำหรับคำสำคัญภายในเครื่องหมายอัญประกาศ
- ใช้ — (ขีดยาว) สำหรับการอธิบายเพิ่มเติม
- ใช้ ... (จุดไข่ปลา) สำหรับข้อความที่ค้างอยู่

### ภาษาเขียน

- ใช้ภาษาเขียนทางการ ไม่ใช้ภาษาพูด
- หลีกเลี่ยงคำสแลง คำวัยรุ่น
- ใช้คำศัพท์เทคนิคเฉพาะเมื่อจำเป็น พร้อมอธิบาย

### ศัพท์เทคนิค

| คำภาษาอังกฤษ | คำภาษาไทย | ใช้เมื่อ |
|---|---|---|
| Concept | มโนทัศน์ | ใช้ "มโนทัศน์" ในเนื้อหาหลัก |
| Theory | ทฤษฎี | ใช้ "ทฤษฎี" ในเนื้อหาหลัก |
| Framework | กรอบแนวคิด | ใช้ "กรอบแนวคิด" ในเนื้อหาหลัก |
| Paradigm | กระบวนทัศน์ | ใช้ "กระบวนทัศน์" ในเนื้อหาหลัก |
| Epistemology | ญาณวิทยา | ใช้ "ญาณวิทยา" ในเนื้อหาหลัก |

## Content Structure (โครงสร้างเนื้อหา)

### Article Structure

```text
1. Identity Block
   - หัวข้อหลัก
   - คำโปรย (1-2 ประโยค)
   - ระดับความลึกซึ้ง
   - เวลาที่ใช้อ่าน

2. Content Block
   - เนื้อหาหลัก
   - หัวข้อย่อย
   - ตัวอย่าง
   - ภาพประกอบ

3. Evidence Block
   - แหล่งอ้างอิง
   - บรรณานุกรม

4. Navigation Block
   - ความรู้ที่เกี่ยวข้อง
   - ปุ่มอ่านต่อ
```

### Concept Structure

```text
1. Identity Block
   - ชื่อมโนทัศน์
   - ชื่อภาษาอังกฤษ
   - รากศัพท์

2. Context Block
   - สำนักคิดที่สังกัด
   - นักคิดที่เกี่ยวข้อง
   - ยุคประวัติศาสตร์

3. Content Block
   - คำนิยาม
   - คำอธิบายเชิงลึก
   - ตัวอย่างจริง

4. Relationship Block
   - มโนทัศน์ที่เกี่ยวข้อง
   - Backlinks

5. Evidence Block
   - แหล่งอ้างอิง

6. Navigation Block
   - ปุ่มอ่านต่อ
   - สำรวจลึกลงไป
```

## Copy Compliance Checklist

- [ ] น้ำเสียงเป็นทางการ อบอุ่น สุขุม
- [ ] ไม่ใช้ภาษาการตลาดฉาบฉวย
- [ ] ใช้ภาษาไทยเป็นหลัก
- [ ] คำศัพท์เทคนิคอธิบายเมื่อจำเป็น
- [ ] ข้อความ error ช่วยเหลือผู้ใช้
- [ ] Empty states เชิญชวนให้สำรวจต่อ
- [ ] Placeholders ชัดเจน
- [ ] Tooltips ให้ข้อมูลที่เป็นประโยชน์
- [ ] วรรคตอนถูกต้องตามมาตรฐานภาษาไทย
