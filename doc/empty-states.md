# ARCHRON Empty States — Phase 05

> **วัตถุประสงค์**: ข้อความเมื่อไม่มีข้อมูล (ทุก context)
> **อ้างอิง**: EDS.md §8 Navigation: "ความรู้ไม่ควรมีจุดสิ้นสุด"

---

## Philosophy (ปรัชญา)

> **"Knowledge should never end."**
> ความรู้ไม่ควรมีจุดสิ้นสุด

Empty states ไม่ใช่จุดจบ แต่เป็นจุดเริ่มต้นของการสำรวจใหม่  
ทุก empty state ต้องเชิญชวนให้ผู้ใช้ค้นหาความรู้ต่อ ไม่ใช่ปิดกั้น

---

## Empty State Patterns (รูปแบบสถานะว่าง)

### 1. Search Not Found (ค้นหาไม่พบ)

│    │  [Icon]      │  │  [Icon]      │  │  [Icon]      │        │
│    │  จิตวิทยา    │  │  ปรัชญา      │  │  ประวัติศาสตร์│        │
│    └──────────────┘  └──────────────┘  └──────────────┘        │

## Empty State Rules (กฎสถานะว่าง)

### Do's

1. **เชิญชวนให้สำรวจต่อ** — ทุก empty state ต้องมี CTA นำไปสู่เนื้อหาอื่น
2. **ให้ข้อมูลที่เป็นประโยชน์** — อธิบายว่าทำไมถึงว่าง และจะทำอย่างไรต่อ
3. **แสดงหมวดหมู่ที่เกี่ยวข้อง** — แนะนำเนื้อหาที่ผู้ใช้อาจสนใจ
4. **รักษาแบรนด์** — ใช้โทนเสียงอบอุ่น สุขุม ชวนขบคิด
5. **响应式** — empty states ต้องทำงานได้ดีบนทุกหน้าจอ

### Don'ts

1. **อย่าปล่อยว่างเปล่า** — ต้องมีข้อความและ CTA เสมอ
2. **อย่าโทษผู้ใช้** — อย่าใช้ข้อความแบบ "คุณพิมพ์ผิด"
3. **อย่าใช้ภาษา technical** — อธิบายให้ผู้ใช้ทั่วไปเข้าใจ
4. **อย่าใช้ไอคอนที่ไม่สื่อความหมาย** — ใช้ไอคอนที่สื่อถึงสถานการณ์
5. **อย่าเกิน 2-3 ประโยค** — กระชับ ได้ใจความ

## Empty State Implementation (การใช้งาน)

### Icons (ไอคอน)

| สถานการณ์ | ไอคอน | Material Symbols |
|---|---|---|
| Search not found | 🔍 | search_off |
| No content | 📚 | auto_stories |
| Empty list | 📖 | library_books |
| Error | ⚠️ | error_outline |
| No related | 🔗 | hub |
| No bookmarks | 🔖 | bookmark_border |

### Colors (สี)

| องค์ประกอบ | สี | Token |
|---|---|---|
| Icon | Muted | --color-muted |
| Heading | Primary text | --color-ink |
| Description | Secondary text | --color-soft-ivory |
| CTA | Accent | --cosmology-accent |
| Background | Paper | --color-paper |

### Spacing (ระยะห่าง)

| องค์ประกอบ | ระยะห่าง |
|---|---|
| Container padding | 48-64px (spacing-8 to spacing-9) |
| Icon to heading | 16-24px (spacing-4 to spacing-5) |
| Heading to description | 8-16px (spacing-2 to spacing-4) |
| Description to CTA | 24-32px (spacing-5 to spacing-6) |
